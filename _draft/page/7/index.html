class Solution:
    def longestPalindrome(self, s: str) -> int:
        dic = {}
        for c in s:
            if c in dic:
                dic[c] += 1
            else:
                dic[c] = 1
        res = 0
        for _, v in dic.items():
            res += v - v % 2
        return res if res == len(s) else res + 1

TCP、UDP、IP协议分别属于什么层？

  • tcp和udp 协议属于传输层
  • IP协议属于网络层

网络中传输层有什么作用？它有那些协议？

  • 主机不同进程间通信。传输层负责在在主机之间提供可靠的数字传输服务。通过流量控制、分段/重组和差错控制，确保数据在网络中正确无误地传输
  • 常见的协议有：TCP、UDP

简述HTTP1.0、1.1、2.0的区别

  • 1.0：支持多媒体对象
  • 1.1：支持长连接、管道化、缓存处理、断点传输
  • 2.0：性能提升、多路复用、头部压缩、服务端推送
