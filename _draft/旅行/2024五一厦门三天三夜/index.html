交通

抵达：厦门北站

景点

鼓浪屿

去鼓浪屿的船票需要提前购买，一般提早半个月就可以购买，

支付宝（或者官网、微信也可以买）搜索登陆“厦门轮渡有限公司”按照提示购买即可。

一般路线是“厦门国际邮轮中心（邮轮中心厦鼓码头）”-“三丘田码头”，票价35元一张，20分钟一趟船，船票包往返，返程不需要再购票，且返程时间可随意安排，购票后刷身份证进入即可（返程也在三丘田坐船）。

厦门大学

提前3天预约

行程

day1：

猫街 -> 南普陀 -> 五老峰 -> 厦门大学（不一定能约上，可以去园林植物园）-> 白城沙滩 -> 演武大桥 -> 沙坡尾 -> 中山路(逛&吃)

day2：

山海健康步道(开端的拍摄地) -> 国际邮轮中心(10：50-13：00) -> 鼓浪屿(18点后再坐船，看夜景)

美食

开禾路口附近：

【明月虾面】：招牌虾面，一碗20元

【浮屿大同鸭肉粥】：原本开在明月虾面隔壁，现在搬迁至（离明月虾面5分钟路程）小学路31号18-1，按料算钱。

【阿杰五香】：五香条是厦门的特色，一根5元

【朱记手撕鸡】：没吃过，但是一直有被推荐

【老二市口土笋冻】：厦门特色小吃，游客不一定能适应，一颗2元，10颗起卖

【阿吉仔】：买特产回家一定一定一定不能错过，绿豆饼和马蹄酥是最特色的，一盒10多元

【芋包王】：小红书看到专门去吃了，不推荐，不好吃。另外，芋包这东西作为本地人我是不喜欢的。

四里附近：

原四里片区于2021年6月拆了，不过原四里有两个美食倒是搬的不远，两百米左右吧。

【四里沙茶面】：强推荐，沙茶面，我的最爱，吃了十几年了，按料算钱，一定要点豆腐！

【四里明香冬粉鸭】：从6元吃到20元，也是开了十几年的老店了，点咸水鸭鸭肉。

【阿忠食府】：距离沙茶面50米的老牌大排档，在附近的可以试试，生意火爆。

中山路附近：

【八婆婆烧仙草】：连锁店，在轮渡码头对面地下通道出来有个店，人较少，中山路口店经常排队

【思西鸡蛋汉堡】：高德可以搜到，二十几年老店，从路边摊到店面，真的好吃，但排队很长

【老思西沙茶烤肉】：点一份沙茶烤肉，15元

【莲欢海蛎煎】：厦门特色海蛎煎，他们家做的还可以，但厦门海蛎煎遍地都是，所以不能说是最好吃的，就是在中山路可以顺路吃吃。另外海蛎煎的做法常见的有三四种，台湾海蛎煎又是另一种做法，商业街的海蛎煎千万别买！

【局口拌面】：拌面和自选汤是他们的特色，不太推荐点鸭肠

【阿卿春卷】：局口街上的小摊位，高德可以搜到

【黄泽和花生汤】：小时候3元一碗，现在也贵了，进去就点花生汤就可以，其他的都已经商业化不那么好吃了

【海滨大厦24楼必胜客餐厅】：估计是全球最高的pizza hat，到这里不是来吃披萨，而是这里可以看到鼓浪屿全景！点一份下午茶看鼓浪屿或者专门上来拍照都行。

【知真味豪香里脊肉串】：多人推荐，未吃过

【月华沙茶面】：不错，没去四里吃可以试试这个

其他美食：

【咏蛙田鸡】：连锁店，中山路、文灶、火车站都有，厦门超超超级好吃的田鸡，强推荐！干锅田鸡、微辣（他们微辣比较辣）

【202海鲜大排档】：连锁大排档，海鲜饼和虾沙拉（切记不要点虾排（趴）沙拉）必点，其他就看各位心情，黑蒜汤的口味大家不一定能适应（比如我）。

【小眼镜海鲜大排档】：海鲜新鲜，口味确实不错，不过菜码太少，是同等价位0.7~0.8倍。大家按推荐菜点就行，这家大排档是以海鲜起家，所以海鲜都很好。

【味友】：连锁店，岛内貌似只有轮渡和火车站有，鸭肉面线（强推）、海蛎炸、椒盐酥皮虾必点，芭乐汁（罐装，红芭乐和芭乐均可）另外仙景芋头很多北方人可能没吃过可以点，其他就不要点了，不好吃，如果只吃一个，就吃鸭肉面线吧。

【临家闽南菜】：对我而言实在有点贵了，尝过原味土猪汤（85元）、小锅萝卜饭（59元），味道还行，但性价比确实不高，里面能吃到的厦门小吃其他地方也能吃到。
